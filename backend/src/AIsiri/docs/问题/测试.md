我帮你整理了一份 **用户可能输入的测试案例库**，覆盖不同场景和边界情况，每个案例都包含：

1. **用户输入**
2. **AI秘书的预期反应**（涉及任务存储 / 调度 / 情绪安慰 / MCP 工具 / 记忆调用等）

---

# 📋 AI秘书用户输入测试案例库

## 🟢 A. 日常任务管理

1. **用户：** “提醒我明天上午10点开会。”

   * 预期：存入 Task DB → Planner 调整明日安排 → 回复 “好的，我已经把明天上午10点的会议记下来了。”

2. **用户：** “我今天晚上要和朋友聚餐。”

   * 预期：Task DB 新增任务 → Planner 检查冲突 → 回复带调度结果。

3. **用户：** “下周一到周三我都要出差。”

   * 预期：批量写入 Task DB → 提示已屏蔽掉这几天其他安排。

4. **用户：** “我等下去拿个快递，大概三点钟。”

   * 预期：存任务 + 调度 → 同时自然回复 “我帮你记好了。”

---

## 🟡 B. 情绪 + 任务混合

5. **用户：** “我好累啊，但还得写报告。”

   * 预期：任务存储 → Planner 排进今天 → 情绪安慰 “别担心，你已经很努力了，我帮你把报告安排在今晚比较空的时间。”

6. **用户：** “我压力好大，明天还有个重要的考试。”

   * 预期：Task DB 存储考试 → Planner 检查日程 → 情绪安慰 + 提供复习建议。

---

## 🔵 C. 纯情绪对话

7. **用户：** “今天真的过得一团糟。”

   * 预期：触发情绪安慰模块 → 长期记忆检索（过去积极事件） → 回复 “我能理解，记得你之前也熬过类似的情况，这次你也会挺过去的。”

8. **用户：** “最近我一直睡不好。”

   * 预期：情绪安慰 → 如果有长期记忆“用户常熬夜” → 回复个性化建议。

---

## 🟣 D. 工具调用 (MCP)

9. **用户：** “我等下要去虹桥机场，现在过去要多久？”

   * 预期：调用高德 MCP → 返回实时路程和时间 → 回复 “现在过去大概需要45分钟，建议你提前出发。”

10. **用户：** “今天上海天气怎么样？”

    * 预期：调用天气 API → 回复当前天气 + 建议。

11. **用户：** “帮我查一下我公司附近的咖啡店。”

    * 预期：调用高德搜索 MCP → 返回候选列表 → 自然回复推荐。

---

## 🟤 E. 长期记忆场景

12. **用户：** “你还记得我最喜欢的电影是什么吗？”

    * 预期：长期记忆检索 → 返回之前存储的偏好。

13. **用户：** “我上个月是不是有一次和小李的聚餐？”

    * 预期：检索长期记忆 → 回复确认。

14. **用户：** “我叫什么名字？”

    * 预期：从长期记忆读取用户姓名 → 正确回答。

---

## ⚫ F. 临界测试 (异常/模糊输入)

15. **用户：** “帮我安排一下最近所有的事吧。”

    * 预期：Planner 汇总 Task DB → 返回简要计划。

16. **用户：** “等下有个事情要做。”（模糊）

    * 预期：模型澄清 “你想让我帮你记录一下具体是什么事情吗？”

17. **用户：** “今天要不要加班呢？”（主观）

    * 预期：情绪安慰 + 调度参考 → 回复 “今天的工作量我帮你看过了，不一定需要加班，你要不要先完成重点任务再看情况？”

18. **用户：** “我太累了，什么都不想做。”

    * 预期：情绪安慰 → 不安排新任务 → 提醒适度休息。

19. **用户：** “我下午三点要开会，四点要拿快递，五点还要去健身，能排得开吗？”

    * 预期：多任务冲突检测 → Planner 输出完整行程表 → 回复 “完全可以，快递我帮你安排在会议结束后，健身照常。”

20. **用户：** “能不能帮我总结一下这周的安排？”

    * 预期：查询 Task DB → 生成周总结 → 输出自然语言。
